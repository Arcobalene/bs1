# Инструкция для Windows 11

## Быстрый старт

### Вариант 1: Автоматический запуск (рекомендуется)

1. **Установка зависимостей:**
   - Дважды кликните на файл `install.bat`
   - Дождитесь завершения установки

2. **Запуск сервера:**
   - Дважды кликните на файл `start.bat`
   - Сервер запустится автоматически

3. **Откройте браузер:**
   - Перейдите по адресу: `http://localhost:3000`

### Вариант 2: Через PowerShell

1. Откройте PowerShell в папке проекта:
   - Нажмите `Shift + ПКМ` в папке проекта
   - Выберите "Открыть окно PowerShell здесь"

2. Установите зависимости:
```powershell
npm install
```

3. Запустите сервер:
```powershell
npm start
```

4. Откройте браузер: `http://localhost:3000`

## Установка Node.js (если не установлен)

1. Перейдите на https://nodejs.org/
2. Скачайте **LTS версию** (рекомендуется)
3. Запустите установщик
4. Следуйте инструкциям установщика
5. Перезапустите PowerShell/CMD после установки

## Проверка установки Node.js

Откройте PowerShell и выполните:
```powershell
node --version
npm --version
```

Если команды работают, Node.js установлен правильно.

## Решение проблем

### Ошибка: "npm не является внутренней или внешней командой"

**Решение:** Node.js не установлен или не добавлен в PATH
1. Переустановите Node.js
2. При установке убедитесь, что выбрана опция "Add to PATH"
3. Перезапустите терминал

### Ошибка: "Порт 3000 уже занят"

**Решение:** Другой процесс использует порт 3000
1. Закройте другие приложения, использующие порт 3000
2. Или измените порт в `server.js` (строка 9)

### Ошибка при установке зависимостей

**Решение:**
1. Запустите PowerShell от имени администратора
2. Выполните: `npm install`
3. Если не помогло, попробуйте: `npm install --force`

### Файл .bat не запускается

**Решение:**
1. Правой кнопкой на файл → "Запуск от имени администратора"
2. Или используйте PowerShell/CMD вручную

## Структура файлов

- `install.bat` — автоматическая установка зависимостей
- `start.bat` — автоматический запуск сервера
- `server.js` — основной файл сервера
- `package.json` — конфигурация проекта

## Остановка сервера

В окне PowerShell/CMD нажмите: `Ctrl + C`

## Полезные команды

```powershell
# Проверка версии Node.js
node --version

# Проверка версии npm
npm --version

# Установка зависимостей
npm install

# Запуск сервера
npm start

# Просмотр установленных пакетов
npm list
```

# Быстрый старт

## 1. Установка зависимостей

```bash
npm install
```

## 2. Запуск сервера

```bash
npm start
```

Сервер запустится на `http://localhost:3000`

## 3. Первые шаги

1. **Создайте аккаунт**: Перейдите на `http://localhost:3000/register`
2. **Войдите в систему**: `http://localhost:3000/login`
3. **Настройте салон**: В админ-панели добавьте услуги и мастеров
4. **Скопируйте ссылку**: В админ-панели будет ваша персональная ссылка для клиентов
5. **Откройте ссылку**: Клиенты могут записываться по вашей ссылке

## Структура

- `/` - Главная страница (нужен userId в URL)
- `/login` - Вход для владельцев
- `/register` - Регистрация нового салона
- `/admin` - Админ-панель (требует входа)
- `/calendar` - Календарь записей (требует входа)

## Важно

- Данные сохраняются в папке `data/` в JSON файлах
- Каждый пользователь имеет свой набор услуг и мастеров
- Пароли хранятся в зашифрованном виде

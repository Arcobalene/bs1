Интеграция с Telegram ботом

Настройте интеграцию с Telegram ботом для получения уведомлений о новых записях.

Получите токен у @BotFather в Telegram. Создайте бота командой /newbot

ID чата или канала, куда будут отправляться уведомления. Узнайте у @userinfobot

Уведомления будут отправляться только если номер телефона клиента совпадает с этим номером. Оставьте пустым, чтобы отправлять уведомления для всех записей.

При включении уведомления будут отправляться при каждой новой записи

Инструкция по настройке:

  1. Откройте Telegram и найдите бота @BotFather
  2. Отправьте команду /newbot и следуйте инструкциям
  3. Скопируйте полученный токен и вставьте в поле "Токен Telegram бота"
  4. Для получения Chat ID найдите бота @userinfobot и отправьте ему любое сообщение
  5. Скопируйте ваш ID (число) и вставьте в поле "Chat ID для уведомлений"
  6. Для каналов: добавьте бота в канал как администратора, затем отправьте сообщение в канал и получите ID через @userinfobot

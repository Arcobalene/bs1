# Учетные данные для входа

## Демо-аккаунт (создается автоматически)

**Логин:** `admin`  
**Пароль:** `admin123`

### Как использовать:

1. Запустите сервер (через Docker или локально)
2. Откройте браузер: `http://localhost:3000/login`
3. Введите логин и пароль выше
4. После входа вы попадете в админ-панель

### Если не получается войти:

**Вариант 1: Автоматическое создание**
- Просто перезапустите сервер - демо-аккаунт создастся автоматически, если его нет

**Вариант 2: Принудительное создание (Windows)**
- Дважды кликните на файл `create-admin.bat`
- Это создаст/пересоздаст демо-аккаунт

**Вариант 3: Через командную строку**
```bash
node create-admin.js
```

### Важно:

- Демо-аккаунт создается автоматически при запуске сервера, если его нет в базе
- Если вы уже создали других пользователей, демо-аккаунт все равно будет создан (если его нет)
- Рекомендуется **сменить пароль** после первого входа или создать свой аккаунт через `/register`

## Создание своего аккаунта

Вы также можете создать свой аккаунт:

1. Перейдите на `http://localhost:3000/register`
2. Заполните форму регистрации
3. После регистрации вы автоматически войдете в систему

## Ссылка для клиентов

После входа в админ-панель вы увидите вашу персональную ссылку для записи клиентов, например:
```
http://localhost:3000/?userId=1
```

Скопируйте эту ссылку и отправьте клиентам для записи.

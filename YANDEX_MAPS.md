# Интеграция с Яндекс.Картами

## Настройка API ключа

Для работы карт необходимо получить API ключ от Яндекс.Карт:

1. Перейдите на https://developer.tech.yandex.ru/
2. Создайте аккаунт или войдите
3. Создайте новый ключ для JavaScript API
4. Скопируйте ключ

## Установка ключа

### Вариант 1: В файлах (для разработки)

Замените `YOUR_API_KEY` в следующих файлах:
- `views/admin.html` (строка с `<script src="https://api-maps.yandex.ru/...`)
- `views/index.html` (строка с `<script src="https://api-maps.yandex.ru/...`)

### Вариант 2: Через переменную окружения (рекомендуется для продакшн)

1. Создайте файл `.env`:
```
YANDEX_MAPS_API_KEY=ваш_ключ_здесь
```

2. Обновите `server.js` для чтения переменной окружения
3. Передавайте ключ в шаблоны через переменные

## Функционал

### В админ-панели:
- Редактирование названия салона
- Указание адреса салона
- Выбор местоположения на карте (клик по карте)
- Сохранение координат

### На главной странице:
- Отображение названия салона в заголовке
- Отображение адреса
- Интерактивная карта с меткой местоположения
- Карта показывается только если указаны координаты

## Без API ключа

Если API ключ не указан:
- Форма редактирования салона все равно работает
- Название и адрес сохраняются
- Карта не отображается (показывается сообщение)

## Бесплатный лимит

Яндекс.Карты предоставляют бесплатный лимит запросов:
- До 25,000 запросов в день
- Достаточно для большинства сайтов

## Альтернатива

Если не хотите использовать Яндекс.Карты, можно:
1. Убрать скрипты карт
2. Оставить только текстовый адрес
3. Или интегрировать Google Maps (требует другой API ключ)
